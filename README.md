# 📊 Mobile Sales Dashboard | Power BI Project

## 👋 About This Project

This project showcases a **Mobile Sales Dashboard** built using **Power BI** to analyze sales performance across different **cities, brands, payment methods, and time periods**.

The goal of this dashboard is to turn raw sales data into **meaningful insights** that help businesses understand performance, customer behavior, and trends.

---

## 🎯 Project Objective

The objective of this project is to:

* Track overall sales performance
* Identify top-performing brands
* Analyze sales by city
* Understand customer ratings
* Monitor payment trends
* Track monthly sales performance

---

This dashboard includes the following key insights:
