# 📊 Power BI Mobile Sales Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes mobile sales performance across different cities, brands, and payment methods.

## 📊 Dashboard Features
- Total Sales KPI
- Total Quantity
- Customer Ratings
- Sales by City (Map)
- Sales by Brand
- Sales by Mobile Model
- Payment Method Analysis

## 🛠 Tools Used
- Power BI
- SQL
- Excel
