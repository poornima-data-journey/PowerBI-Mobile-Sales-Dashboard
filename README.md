## 🚀 Project Overview

This **Power BI Mobile Sales Dashboard** provides comprehensive insights into mobile sales performance across different **cities, brands, payment methods, and time periods**.

The dashboard helps stakeholders analyze business performance, identify trends, and make data-driven decisions.

--

## 🎯 Business Problem

Businesses need to monitor:

* Sales performance
* Customer behavior
* Brand performance
* Payment trends
* Regional sales distribution

This dashboard solves these problems by providing an **interactive and dynamic visualization**.

## 📈 Key Insights

* Apple leads in total sales
* Sales peak in July
* Debit Card and Cash are most used payment methods
* Metro cities generate highest revenue
* Weekend sales are higher than weekdays
